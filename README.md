# gcam-config
This is the GCam config for any modded GCam you wish to use in your Android 12+ phones.
(Only works with gcam series 7 / version 7> and >8 (example- 7.2, 7.3 ,7.4, etc.)

The configs are made using Urynx05 GCam 7.3.018_2.6 latest Series 7 urynx gcam available.
